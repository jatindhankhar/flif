## Instructions
<ul>
  <li> Install Ruby </li>
  <li> Install bundler `gem install bundler` </li>
  <li> `bundle install` to install Dependencies </li>
  <li> To test locally `rackup config.ru` </li>
  <li> To run on production `bash restart_server.sh`</li>
</ul>
